# Astronaut Daily Schedule

## Project Idea
This project shows a simple daily schedule of astronauts aboard the International Space Station (ISS). It educates users about how astronauts organize their day in space, including exercise, experiments, meals, and sleep.  
It helps NASA by promoting educational outreach and building awareness about astronaut life.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Clone or download the repository.
2. Open `index.html` in any web browser.
